# one-ayx-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能涉及各类技术演示、文档预览、工具界面或静态内容展示，不针对任何特定域名或网站。

## 项目简介

本仓库旨在以静态 HTML 文件的形式，收集和整理一系列独立的前端页面。所有页面均为自包含的 HTML 文件（可能附带 CSS、JavaScript 或资源链接），可直接在浏览器中打开查看。

仓库本身不提供后端服务或动态内容生成，仅作为文件存储与版本管理的平台。

## 目录说明

仓库根目录下按页面用途或主题划分为子目录，结构如下：

```
one-ayx-guide/
├── page-group-a/        # 页面组 A（示例）
│   ├── index.html
│   └── assets/
├── page-group-b/        # 页面组 B（示例）
│   ├── index.html
│   └── ...
├── standalone/          # 独立单页
│   └── example.html
└── README.md            # 本文件
```

每个子目录内通常包含一个或多个 HTML 文件，以及所需的静态资源（如图片、样式表、脚本等）。具体内容请参考各子目录内的说明或文件注释。

## 页面归档说明

归档的页面可能包含但不限于以下类型：
- 技术演示或原型
- 文档或笔记的 HTML 版本
- 静态工具界面
- 学习或测试用页面

所有页面均以原始 HTML 形式存储，不进行压缩或混淆，方便审阅和修改。页面内容可能引用外部资源（如公共 CDN 库），但不依赖任何特定的外部服务。

## 维护说明

- 本仓库由个人维护，不定期更新。
- 欢迎通过 Issue 或 Pull Request 提交改进建议、问题报告或新增页面。
- 提交新页面时，请遵循以下原则：
  - 文件放入对应主题的子目录，或新增目录并附带简短说明。
  - 确保页面为自包含或明确说明依赖的外部资源。
  - 避免包含恶意代码、隐私收集或商业推广内容。
- 仓库不保证所有页面长期可用或兼容最新浏览器版本。
